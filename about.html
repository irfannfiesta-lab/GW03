System Architecture & Paradigms

An in-depth explanation of TBR, CBR, and ABR in Multimedia Database Systems.

1. Overview of Retrieval Paradigms

In multimedia databases, retrieving images is fundamentally different from retrieving text records. ChromaSeek employs three major paradigms to bridge the semantic gap:

ParadigmFull NameKey Idea
TBRText-Based RetrievalSearch by keywords, tags, captions, or metadata annotated by humans.
CBRContent-Based RetrievalSearch by the visual content itself — colors, textures, shapes, spatial layout.
ABRAttribute-Based RetrievalSearch by mid-level semantic attributes — structured properties derived from content (e.g., "dominant color = blue").

2. Text-Based Retrieval (TBR)

TBR is the oldest and simplest paradigm. Every image in the database is manually annotated with textual metadata. Retrieval reduces to a text search problem. TBR does not analyze the image pixels at all.

Mathematical Formulation

Given a query string Q and documents D, we tokenize them and compute TF-IDF vectors:

TF(w, tᵢ) = (frequency of w in tᵢ) / (total terms in tᵢ) IDF(w) = log(N / df(w)) where N = total documents, df(w) = documents containing w TF-IDF(w, tᵢ) = TF(w, tᵢ) × IDF(w)

We then compute the Cosine Similarity between the query vector and document vector to find the closest matches:

sim(q, tᵢ) = (q⃗ · t⃗ᵢ) / (‖q⃗‖ × ‖t⃗ᵢ‖)

3. Content-Based Retrieval (CBR)

CBR analyzes the actual pixel content of images, extracting color histograms and statistical moments to bridge the "semantic gap" without relying on human labels.

Mathematical Formulation — Color Features

Color Histogram

H = [h₁, h₂, …, h_B] where hⱼ = (number of pixels in bin j) / (total pixels)

Color Moments (Statistical Descriptors)

Mean (1st moment): μ_c = (1/N) × Σᵢ pᵢ,c Std Dev (2nd moment): σ_c = √[(1/N) × Σᵢ (pᵢ,c − μ_c)²] Skewness (3rd moment): s_c = ∛[(1/N) × Σᵢ (pᵢ,c − μ_c)³]

Distance Metrics for CBR

We use the Chi-Square Distance which is optimal for histogram comparisons as it normalizes the differences:

χ²(Q, D) = Σⱼ (Qⱼ − Dⱼ)² / (Qⱼ + Dⱼ)

4. Attribute-Based Retrieval (ABR)

ABR sits between TBR and CBR. It extracts structured, named attributes from image content (e.g. "Warm", "High Saturation"), converting low-level features into mid-level semantic properties.

Mathematical Formulation

1. Dominant Color Extraction (K-Means)

Objective: minimize J = Σᵢ₌₁ᴺ Σₖ₌₁ᴷ rᵢₖ ‖pᵢ − μₖ‖² where rᵢₖ = 1 if pixel i belongs to cluster k, 0 otherwise μₖ = cluster center (the "dominant color")

2. Colorfulness Metric (Hasler & Süsstrunk)

rg = R − G yb = 0.5 × (R + G) − B Colorfulness = √(σ_rg² + σ_yb²) + 0.3 × √(μ_rg² + μ_yb²)

3. Attribute Matching — Weighted Distance

dist(A_q, A_d) = Σⱼ wⱼ × δⱼ(A_q[j], A_d[j])
